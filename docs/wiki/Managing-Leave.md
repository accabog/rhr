# Managing Leave

This guide covers requesting time off, checking balances, and approving leave requests.

## Leave Balances

View your current leave balances from:
- The **Dashboard** (summary view)
- **Leave > Balances** (detailed breakdown)

Balances show:
- **Available** - Days you can currently use
- **Used** - Days taken this period
- **Pending** - Days in pending requests

## Requesting Leave

### Submit a Request

1. Navigate to **Leave > Requests**
2. Click **New Request**
3. Fill in the form:
   - **Leave Type** - PTO, sick leave, etc.
   - **Start Date** - First day of leave
   - **End Date** - Last day of leave
   - **Notes** - Optional reason or details
4. Click **Submit**

Your request will be sent to your manager for approval.

### Request Status

Requests have the following statuses:

| Status | Meaning |
|--------|---------|
| Pending | Awaiting manager approval |
| Approved | Request approved, leave confirmed |
| Rejected | Request denied (see comments for reason) |
| Cancelled | You cancelled the request |

### Cancel a Request

You can cancel pending or approved requests:
1. Go to **Leave > Requests**
2. Find your request
3. Click **Cancel**
4. Confirm cancellation

Note: Cancelling approved leave may require manager notification.

## Calendar View

The **Calendar** shows:
- Your scheduled leave
- Team members' leave (if permitted)
- Company holidays

Use this to plan time off around team availability.

## For Managers: Approving Leave

If you manage a team, you'll see pending requests in your dashboard.

### Review Requests

1. Go to **Dashboard** or **Leave > Approvals**
2. Click on a pending request
3. Review the details:
   - Employee's current balance
   - Team calendar for conflicts
   - Request notes
4. Click **Approve** or **Reject**
5. Add comments if rejecting

### Best Practices

- Review requests promptly
- Check team calendar for coverage
- Add a reason when rejecting to help the employee reschedule
