# Getting Started

This guide covers initial setup and basic navigation for new Raptor HR users.

## First Login

1. You'll receive an email invitation from your HR administrator
2. Click the link to set your password
3. Log in with your email and password

## Dashboard

After logging in, you'll see your personal dashboard with:

- **Upcoming leave** - Your scheduled time off
- **Leave balances** - Available PTO, sick days, etc.
- **Recent time entries** - Your logged hours
- **Pending approvals** (managers only) - Items awaiting your review

## Navigation

The main menu provides access to:

| Section | Description |
|---------|-------------|
| Dashboard | Overview and quick stats |
| Employees | Employee directory and profiles |
| Leave | Request time off and view balances |
| Time Tracking | Log daily work hours |
| Timesheets | Submit weekly/monthly timesheets |
| Calendar | Team leave and company holidays |

## Your Profile

Click your name in the top-right corner to access:

- **Profile** - Update your contact information and preferences
- **Settings** - Notification preferences
- **Logout** - Sign out of the system

## Common Tasks

### Requesting Leave

1. Go to **Leave** in the main menu
2. Click **New Request**
3. Select leave type, dates, and add notes
4. Submit for approval

### Logging Time

1. Go to **Time Tracking**
2. Click **Add Entry** or use the timer
3. Select the entry type and enter hours
4. Save your entry

See [Managing Leave](Managing-Leave) and [Time Tracking](Time-Tracking) for detailed guides.
