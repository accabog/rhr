# Time Tracking

This guide covers logging work hours and submitting timesheets.

## Daily Time Entries

### Adding Time Entries

1. Navigate to **Time Tracking**
2. Click **Add Entry**
3. Fill in:
   - **Date** - The work date
   - **Entry Type** - Regular, overtime, etc.
   - **Hours** - Time worked
   - **Notes** - Optional description of work
4. Click **Save**

### Using the Timer

For real-time tracking:
1. Click **Start Timer** to begin
2. Work on your task
3. Click **Stop** when finished
4. Review and save the entry

### Editing Entries

1. Find the entry in your list
2. Click to edit
3. Update the details
4. Save changes

Note: You may not be able to edit entries in submitted timesheets.

## Timesheets

Timesheets group your time entries for a period (weekly or monthly) for manager approval.

### Submitting a Timesheet

1. Go to **Timesheets**
2. Select the period to submit
3. Review your entries for accuracy
4. Click **Submit**

Once submitted, the timesheet goes to your manager for approval.

### Timesheet Status

| Status | Meaning |
|--------|---------|
| Draft | In progress, not yet submitted |
| Submitted | Awaiting manager approval |
| Approved | Manager has approved |
| Rejected | Needs corrections (see comments) |

### Corrections

If your timesheet is rejected:
1. Review the manager's comments
2. Make the requested corrections
3. Resubmit for approval

## For Managers: Reviewing Timesheets

### Approve Timesheets

1. Go to **Timesheets > Approvals**
2. Select a submitted timesheet
3. Review entries for accuracy
4. Click **Approve** or **Reject**
5. Add comments if rejecting

### What to Check

- Total hours align with expectations
- Overtime is properly categorized
- Notes provide sufficient context
- No duplicate or missing entries
